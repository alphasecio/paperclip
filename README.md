## Choosing a platform

**Railway** — One-click deploy, no server management, automatic TLS. Costs ~$7-10/month depending on whether you use embedded or external PostgreSQL.

**DigitalOcean** — Fixed monthly cost, full server control, bring your own domain. Requires `doctl` and costs $12-24/month depending on choice of droplet.
